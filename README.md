# ミルナ - Miluna the Black Cat 🌙✨🎵

ミルナの公式Webサイトへようこそ！銀河と月、そして音楽から名付けられた黒猫の男の子のキャラクターサイトです。

![ミルナ](https://www.genspark.ai/api/files/s/NooiHxkv)

## 🌟 プロジェクト概要

このプロジェクトは、ミルナというキャラクターの魅力を伝えるための遊び心あふれるWebサイトです。宇宙をテーマにしたデザインで、音楽要素を取り入れたインタラクティブなアニメーションと美しいビジュアルエフェクトを特徴としています。

### キャラクター情報

- **名前**: ミルナ (Miluna)
- **性別**: 男の子
- **種族**: 黒猫（ファーキャラクター）
- **名前の由来**: ミルキーウェイ（Milky Way / 銀河）+ ルナ（Luna / 月）+ 音楽（Music）
- **キャッチコピー**: 「〜 銀河のメロディに乗せて 〜 ♪ 星空を奏でる黒猫 ♪」
- **特徴**: 
  - ✨ 額に輝く紫色のストライプ
  - 🌙 大きな白と黒の耳
  - 💙 魅力的なブルーの瞳
  - 🌌 ダークグレーとブラックの毛並み
  - 🎵 音楽が大好きな性格
  - 🐾 思慮深く遊び心のある性格

### 1. **レスポンシブデザイン**
   - デスクトップ、タブレット、スマートフォンに完全対応

   - ヒーローセクションのグリッチエフェクト
   - 月の浮遊アニメーション
   - スクロールに応じたフェードインアニメーション
   - ホバーエフェクト（カード、ボタン、画像など）

### 5. **🎮 遊び心満載のインタラクティブ機能**

### 7. **🎵 音楽要素の統合（NEW！）**
   - 音楽をテーマにした新しいキャッチコピー
   - ストーリーに「音楽との出会い」を追加
   - 特徴セクションに「音楽の才能」を追加
   - 踊る音符の強化エフェクト
   - 音楽関連のアイコンとデザイン

### 8. **充実したコンテンツセクション**
   - **ホーム**: インパクトのあるヒーローセクション
   - **プロフィール**: 詳細なキャラクター情報
   - **ギャラリー**: 画像ギャラリー（ライトボックス対応）
   - **ストーリー**: キャラクターの物語（音楽要素追加）
   - **特徴**: ミルナの魔法の特徴

### 9. **視覚的エフェクト**

## 🎮 インタラクティブ機能の使い方

### 基本操作
1. **マウス移動**: 猫の足跡が残ります
2. **ダブルクリック**: 星座を描けます（5つの点で自動リセット）
3. **「miluna」とタイプ**: 虹のエフェクトが出現
4. **コナミコマンド**: パーティーモードが発動
5. **歩く猫をクリック**: ハートエフェクトが飛び出す
6. **左下の？ボタン**: ヘルプガイドを表示

8. **NEW! 矢印キー**: ライトボックス内で前後の画像に移動

- **クリック**: 画像を拡大表示
- **×ボタン**: ライトボックスを閉じる
- **← / →ボタン**: 前後の画像に移動
- **矢印キー**: キーボードで前後の画像に移動
- **ESCキー**: ライトボックスを閉じる
- **背景クリック**: ライトボックスを閉じる

### SNSリンク
- ハンバーガーメニュー内にSNSアイコン配置
- Twitter、Instagram、Discord、Email へのリンク
- モバイルでもアクセス可能

### デバッグとエラーハンドリング
- **画像読み込みエラー**: 自動的に猫の絵文字プレースホルダーに切り替え
- **コンソールログ**: すべての機能が正常に動作（画像403エラーは既知で対処済み）

## 📝 カスタマイズ方法

### 色の変更
`css/style.css` の `:root` セクションでカラー変数を変更できます：

```css
:root {
    --primary-color: #9b59b6;  /* メインカラー */
    --secondary-color: #3498db; /* サブカラー */
    /* その他の色... */
}
```

### SNSリンクの変更
`index.html` のナビゲーションセクションでSNSリンクを実際のURLに変更：

```html
<a href="https://twitter.com/your_account" target="_blank" title="Twitter">
    <i class="fab fa-twitter"></i>
</a>
```

### コンテンツの追加
`index.html` の各セクションにコンテンツを追加できます。ギャラリーセクションは以下のように簡単に拡張可能：

```html
<div class="gallery-card" data-lightbox="gallery" 
     data-img="画像URL" data-title="タイトル" data-desc="説明">
    <img src="画像URL" alt="説明">
    ...
</div>
```

### アニメーションの調整
`js/main.js` で各アニメーションのタイミングや動作を調整できます。

## 📝 カスタマイズ方法

### 色の変更
`css/style.css` の `:root` セクションでカラー変数を変更できます：

```css
:root {
    --primary-color: #9b59b6;  /* メインカラー */
    --secondary-color: #3498db; /* サブカラー */
    /* その他の色... */
}
```
