# ミルナ - Miluna the Black Cat 🌙✨

ミルナの公式Webサイトへようこそ！銀河と月から名付けられた黒猫の男の子のキャラクターサイトです。

![ミルナ](https://www.genspark.ai/api/files/s/NooiHxkv)

## 🌟 プロジェクト概要

このプロジェクトは、ミルナというキャラクターの魅力を伝えるための遊び心あふれるWebサイトです。宇宙をテーマにしたデザインで、インタラクティブなアニメーションと美しいビジュアルエフェクトを特徴としています。

### キャラクター情報

- **名前**: ミルナ (Miluna)
- **性別**: 男の子
- **種族**: 黒猫（ファーキャラクター）
- **名前の由来**: ミルキーウェイ（Milky Way / 銀河）+ ルナ（Luna / 月）
- **特徴**: 
  - ✨ 額に輝く紫色のストライプ
  - 🌙 大きな白と黒の耳
  - 💙 魅力的なブルーの瞳
  - 🌌 ダークグレーとブラックの毛並み
  - 🐾 思慮深く遊び心のある性格

## 📝 カスタマイズ方法

### 色の変更
`css/style.css` の `:root` セクションでカラー変数を変更できます：

```css
:root {
    --primary-color: #9b59b6;  /* メインカラー */
    --secondary-color: #3498db; /* サブカラー */
    /* その他の色... */
}
```

### コンテンツの追加
`index.html` の各セクションにコンテンツを追加できます。特にギャラリーセクションは簡単に拡張可能です。

### アニメーションの調整
`js/main.js` で各アニメーションのタイミングや動作を調整できます。

## 🌐 デプロイ方法

このサイトを公開するには、**Publishタブ**を使用してワンクリックでデプロイできます！

1. Publishタブを開く
2. 「公開」ボタンをクリック
3. 生成されたURLを共有
